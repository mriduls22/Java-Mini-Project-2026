# Java-Mini-Project-2026
6th Sem BCA Project

🎮 Brick Breaker Game (Java)

A classic Brick Breaker arcade game developed using Java, implementing core concepts of Object-Oriented Programming (OOP), GUI design, and event-driven programming.

This project is designed as a mini project to demonstrate real-time game development using Java Swing and AWT.


🚀 Features
🎯 Interactive paddle control using keyboard
🟡 Smooth ball movement with collision detection
🧱 Dynamic brick patterns (random generation)
❤️ Lives system (Game Over when lives = 0)
📈 Score tracking with high score
🎚️ Level progression with increasing difficulty
⏸️ Pause/Resume functionality
🔁 Restart option after win/lose


🛠️ Technologies Used
Java
Java Swing (GUI)
AWT (Graphics & Events)
OOP Concepts
Event Handling (KeyListener, Timer)


📂 Project Structure
BrickBreaker/
│── Main.java           // Entry point of the game
│── GamePlay.java       // Core game logic & rendering
│── MapGenerator.java   // Brick layout generation


🎮 How to Play
▶️ Press ENTER to Start / Restart the game
⬅️ Press LEFT ARROW to move paddle left
➡️ Press RIGHT ARROW to move paddle right
⏸️ Press P to Pause/Resume

Objective:

Break all the bricks using the ball without letting it fall below the paddle.
